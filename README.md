# 个人网站使用指南

这是一个专为企业界人士设计的个人展示网站，适合用于工作展示和求职。

## 📁 文件结构

```
personal-website/
├── index.html      # 主页面
├── style.css       # 样式文件
├── script.js       # 交互脚本
├── avatar.jpg      # 个人头像（需自行添加）
├── resume.pdf      # 简历PDF（需自行添加）
└── README.md       # 本文件
```

## 🎨 设计特点

- **专业简洁**：参考学术网站风格，但更侧重企业工作展示
- **响应式设计**：自适应手机、平板、电脑屏幕
- **清晰结构**：关于我 → 工作经历 → 项目案例 → 技能 → 联系方式
- **数据导向**：强调用数据展示工作成果
- **易于定制**：所有内容都用 [方括号] 标注，方便替换

## ✏️ 如何定制

### 1. 替换个人信息

在 `index.html` 中搜索所有 `[方括号]` 内容并替换：

- `[您的姓名]` → 你的真实姓名
- `[您的职位]` → 如"产品经理"、"技术总监"
- `[行业领域]` → 如"互联网"、"金融科技"
- 其他所有方括号内容

### 2. 添加头像

- 准备一张正方形头像照片（建议 500x500px）
- 命名为 `avatar.jpg` 放在网站根目录
- 或修改 HTML 中的图片路径

### 3. 填写工作经历

在"工作经历"部分：
- 按时间倒序排列（最新的在最上面）
- 用**数据**说话：增长率、用户量、营收等
- 突出**成果**而非职责

示例：
```
❌ 负责产品设计工作
✅ 主导产品改版，用户留存率提升35%，DAU增长至50万
```

### 4. 展示项目案例

每个项目卡片包含：
- **项目名称**：简洁有力
- **项目类型**：如"产品创新"、"增长项目"
- **背景目标**：为什么做这个项目
- **你的角色**：你具体负责什么
- **关键成果**：用数据证明价值

### 5. 调整配色

在 `style.css` 顶部修改颜色变量：

```css
:root {
    --primary-color: #2c3e50;    /* 主色调（深蓝灰）*/
    --secondary-color: #3498db;  /* 强调色（蓝色）*/
    --accent-color: #e74c3c;     /* 点缀色（红色）*/
}
```

推荐配色方案：
- **科技感**：`#1a1a2e` + `#16213e` + `#0f3460`
- **商务风**：`#2c3e50` + `#34495e` + `#3498db`
- **创意型**：`#6c5ce7` + `#a29bfe` + `#fd79a8`

## 🚀 部署上线

### 方案1：GitHub Pages（免费）

1. 在 GitHub 创建仓库（如 `username.github.io`）
2. 上传所有文件
3. 在仓库设置中启用 GitHub Pages
4. 访问 `https://username.github.io`

### 方案2：Vercel（免费，推荐）

1. 注册 [Vercel](https://vercel.com)
2. 导入 GitHub 仓库或直接拖拽文件夹
3. 自动部署，获得 `xxx.vercel.app` 域名
4. 可绑定自定义域名

### 方案3：自己的服务器

上传到服务器的 Web 目录（如 `/var/www/html`）

## 📝 内容建议

### 关于我部分
- 用 2-3 段话介绍自己
- 第一段：核心定位和年限
- 第二段：主要成就（用数据）
- 第三段：职业目标或特质

### 工作经历
- 只写近 3-5 份工作
- 每份工作 3-5 个要点
- 重点写**成果**，不是职责描述

### 项目案例
- 选 3-6 个最有代表性的项目
- 每个项目说清楚：背景、角色、成果
- 用数据量化影响力

### 技能部分
- 分类展示：核心能力、技术工具、行业知识
- 只写真正掌握的技能
- 可以标注熟练程度

## 🎯 求职优化建议

1. **SEO 优化**：在 HTML `<head>` 中添加：
```html
<meta name="description" content="[你的一句话介绍]">
<meta name="keywords" content="产品经理,互联网,求职">
```

2. **添加 PDF 简历**：
   - 准备一份精美的 PDF 简历
   - 命名为 `resume.pdf` 放在根目录
   - 网站上提供下载按钮

3. **社交媒体链接**：
   - LinkedIn（必须）
   - GitHub（技术岗）
   - 个人博客/公众号

4. **数据追踪**（可选）：
   添加 Google Analytics 了解访问情况
